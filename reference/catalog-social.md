## Capabilities

### tiktok-profile — Get TikTok creator profile
```json
{"handle": "stoolpresidente"}
```

### tiktok-posts — Get TikTok creator posts
```json
{"handle": "stoolpresidente"}
```

### instagram-profile — Get Instagram user profile
```json
{"handle": "zuck"}
```

### reddit-search — Search Reddit posts
```json
{"query": "ai agents"}
```

More social capabilities available for TikTok, Instagram, Facebook, and Reddit — including comments, followers, following, stories, highlights, and search variants.
