# Catalog: Niche Data

## How to pay

```
clawcard agent pay <slug> '<json>' --json
```

## Capabilities

### sneaker-prices ($0.01) — powered by KicksDB
Get sneaker and streetwear prices from StockX, GOAT, and more.
```json
{"query": "jordan 4 retro"}
```

### property-value ($0.04) — powered by RentCast
Get automated property valuation for any US address.
```json
{"address": "123 Main St, San Francisco, CA 94105"}
```

More data capabilities available — rent estimates, property records, IP intelligence, phone validation.

Browse all: `clawcard agent catalog-search "data" --json`
